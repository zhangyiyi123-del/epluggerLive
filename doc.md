一、核心业务需求
用户登录与权限
支持密码登录、验证码登录、企业 SSO 登录三种方式
登录态持久化（localStorage），未登录状态下所有页面重定向至登录页
表单校验（手机号格式、密码 / 验证码非空）及错误提示
四大核心功能模块
运动打卡：支持多类型运动选择、打卡数据录入（时长 / 距离 / 卡路里）、强度选择、佐证图片上传（≤3 张）、目标进度展示、历史记录查看
正向行为打卡：支持行为分类选择、同事 @提及、行为描述、佐证上传、积分奖励预览、历史记录查看
圈子社交：动态列表（筛选 / 分页）、动态发布（富文本 / 多图 / 话题 /@/ 可见范围）、动态详情（点赞 / 收藏 / 评论 / 二级回复）、我的动态管理
积分体系：积分统计（累计 / 已用 / 可用）、等级进度展示、勋章墙（已获得 / 待解锁）、积分商城（商品展示 / 兑换）、排行榜（积分 / 运动 / 正向榜）、消息通知（点赞 / 评论 /@）
个人中心
个人数据统计（连续打卡天数 / 累计积分 / 勋章数）
功能菜单（我的动态 / 我的消息 / 通知设置 / 帮助反馈）
深色模式切换（UI 展示）、退出登录
